# 一些有用的VSCode使用技巧

#### 常用快捷键：

| 快捷键| 作用 |
| ------| ------|
| command + , | 打开用户设置 |
| command+option | 选中|
| option+ 上（下）方向键 |  把该行向上（向下移动）|
| option+shift+F |  格式化|
| ctrl + tab |  切换文件|
| ctrl + h |  打开dash|
| command + p	 |  切换文件（列表选择）|
| command+option+a |  打开carbon|
| command + /	 |  快捷注释|
| ctrl + p — 在冒号中输入行数	 |  行跳转|
